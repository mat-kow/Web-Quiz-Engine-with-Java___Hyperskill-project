# Web Quiz Engine with Java - [Hyperskill project](https://hyperskill.org/projects/91)

This project delivers a fully functional web service for creating and managing quizzes. It supports answering questions, evaluating results, and storing data using an embedded database. The application demonstrates key backend development concepts such as building a REST API, implementing security, and handling persistent data.

Graduation project of Java Backend Developer (Spring Boot) [Course](https://hyperskill.org/courses/12-java-backend-developer-spring-boot)
